As of 22 May 2022, the `master`, `kirkstone`, and `dunfell` branches support L4T R32.7.2/JetPack 4.6.2.

# Changes from L4T R32.7.1/JetPack 4.6.1

NVIDIA documentation:
* [L4T R32.7.2 page](https://developer.nvidia.com/embedded/linux-tegra-r3272) Note that the release notes linked on that page are for R32.7.1 (as of 22 May 2022).
* [JetPack release page](https://developer.nvidia.com/jetpack-sdk-462)

Content for this release is identical to L4T R32.7.1/JetPack 4.6.1, with the following updates:
* [Security fixes](https://nvidia.custhelp.com/app/answers/detail/a_id/5343)
* Support for new [AGX Xavier module revisions](https://developer.nvidia.com/jetson-agx-xavier-pcn208560-bom-addition-dram-and-emmc)
