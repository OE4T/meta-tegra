As of 21 Jan 2023, the `kirkstone-l4t-r32.7.x` and `dunfell` branches support L4T R32.7.3/JetPack 4.6.3.

# Changes from L4T R32.7.2/JetPack 4.6.2

NVIDIA documentation:
* [L4T R32.7.3 page](https://developer.nvidia.com/embedded/linux-tegra-r3273)
* [JetPack release page](https://developer.nvidia.com/jetpack-sdk-463)

Content for this release is identical to L4T R32.7.2/JetPack 4.6.2, with the following updates:
* Security fixes 
* Support for updated [Jetson TX2 modules](https://developer.nvidia.com/jetson-tx2-pcn209140-addition-manufacturing-bom-components-hynix-dram) with Hynix DRAM
